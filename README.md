# # 🎯 Username Hunter — نظام صيد اليوزرات الذكي

مشروع تخرج | أجينت ذكي يفحص يوزرات على Instagram + Snapchat + Twitter

---

## ⚙️ المتغيرات في Render

| المتغير | مطلوب؟ | الوصف |
|---------|--------|-------|
| `WEBHOOK_URL` | ✅ نعم | رابط ديسكورد ويب هوك |
| `DELAY_MIN` | لا (افتراضي 20) | أقل تأخير بين الفحوصات (ثانية) |
| `DELAY_MAX` | لا (افتراضي 45) | أكثر تأخير بين الفحوصات (ثانية) |
| `USERNAME_LEN` | لا (افتراضي 4) | طول اليوزر (3 أو 4) |
| `IG_SESSION` | اختياري | سيشن انستا لنتائج أدق |
| `TW_BEARER` | اختياري | توكن تويتر API |

---

## 🚀 طريقة الرفع على Render

1. ارفع الملفين `main.py` و `requirements.txt` على GitHub
2. أنشئ Web Service جديد على Render
3. أضف المتغيرات في Environment Variables
4. اضغط Deploy ✅

---

## 📡 الـ Endpoints

- `GET /` — حالة البرنامج
- `GET /stats` — إحصائيات تفصيلية

---

## 🧠 كيف يعمل الأجينت

1. يولّد يوزر عشوائي (3-4 خانات)
2. يفحصه على الثلاث منصات **بشكل متوازي**
3. لو منصة Rate Limit، يتجاوزها ويكمل على الباقين
4. لو لقى صيدة، يرسل إشعار @everyone على ديسكورد
5. الداشبورد يتحدث كل 15 ثانية بدون رسايل جديدة
